# Частотный словарь
## Данная программа считает, сколько раз в тексте встречаются слова.

### Сборка:
> make (сборка прописана дефолтной целью в Makefile)

### Запуск
> ./freq <путь до файла на чтение> <путь до файла на запись результата>

*** Файл на чтение нужно создать. Для файла на запись можно просто задать путь (файл создастся сам) ***
